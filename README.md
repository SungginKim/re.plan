# Recipe App

## Overview

This is a full-stack recipe platform developed to improve skills in frontend development, server-side logic, state management, RESTful APIs, and user authentication. Users can store, view, edit, and delete their recipes, providing a simple way to manage favorite meals.

## Features

- **Add Recipes:** Enter name, ingredients, instructions, cooking time, prep time, category, and difficulty level.
- **View Recipes:** List of saved recipes with name, brief description, and optional thumbnail.
- **Edit Recipes:** Update name, ingredients, instructions, or image.
- **Favorites:** Mark recipes as favorite; view them in the favorites section.
- **Search (Optional):** Search by keyword or filter by category.

## Tech Stack

- **Frontend:** React, Tailwind CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JWT-based login

## Goal

Gain hands-on experience in building a full-stack application from scratch with CRUD operations and user authentication.
